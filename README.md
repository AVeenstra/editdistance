# Edit Distance (Levenshtein distance) calculation with a GPU (cluster)

## Installation
### Debian (Jessie)
#### AMD GPUs
Required packages:
 * amd-opencl-dev
 * amd-libopencl1-dev
 * gcc
 * g++
 * cmake
 * amd-opencl-icd
 * amd-clinfo
 * opencl-headers
 * fglrx-driver

### Windows
Required programs:
 * Visual Studio (tested with 2017)
 * Driver of your graphics card

### Compilation
Build with CMake.

### Issues
Make sure the executing user has enough pliviliges
